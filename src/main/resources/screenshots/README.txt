Screenshots
===========

This directory should contain screenshots of the chess application for documentation.

Recommended screenshots to include:
- gameplay.png - A screenshot of an active game session
- login.png - The login/connection screen
- game_over.png - End of game screen showing winner or draw

These images are referenced in the README.md documentation.

For consistency:
- Use PNG format
- Crop images to show only the relevant parts
- Try to keep file sizes reasonable (< 1MB)
