CHESS GAME v1.0.0
==============

This package contains version 1.0.0 of the chess game. You can use the client and server components to play online.

CONTENTS:
---------
1. chess-client-1.0.0.jar - Game Client
2. chess-server-1.0.0.jar - Game Server

REQUIREMENTS:
-------------
- Java 17 or higher

INSTALLATION:
-------------
No installation required. Simply copy the JAR files to your computer.

LAUNCHING:
----------
Client:
    java -jar chess-client-1.0.0.jar

Server (for server administrators):
    java -jar chess-server-1.0.0.jar

HOW TO PLAY:
------------
1. The server must be started first (this will be done by the system administrator)
2. Launch the client program
3. Enter the server address (e.g., localhost or the server IP address)
4. Enter your player name and connect
5. Wait for an opponent or challenge an existing player
6. Drag and drop pieces on the chess board to make your move

GAME RULES:
-----------
- Standard chess rules apply
- Castling, en passant, and pawn promotion are supported
- The game ends with checkmate, stalemate, threefold repetition, or the 50-move rule

TROUBLESHOOTING:
----------------
- Connection issues: Make sure the server address is entered correctly
- Ensure the server is running
- Check your network connection
